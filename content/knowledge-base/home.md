# LNKDREC Knowledge Base

Welcome to the LNKDREC Knowledge Base.

This wiki is the clean, current reference for how the dashboard is structured and how each page behaves.

## How to use this wiki

Before starting work on the platform:

1. Open the relevant dashboard page in this Knowledge Base
2. Read the page description
3. Review the available features on that page
4. Check how each feature behaves, including any important rules or quirks

## What each page will document

Each dashboard section page is intended to describe:

- What the page is for
- Which features are available on the page
- How each feature works and behaves
- Important implementation notes worth remembering

## Dashboard pages covered

- Candidates
- Pipeline
- Sourcing
- Leads
- Jobs
- Client Portal
- Settings
- Analytics

## Documentation model

This system uses two layers:

- **Clean Wiki** — this Knowledge Base, focused on current truth
- **Living Memory** — a separate running log for changes, decisions, and updates over time

The goal is simple: make it easy to review the dashboard before building, debugging, or extending anything.
